# uab-weapp
一个微信小程序组件库

## 下载代码并安装依赖:
```sh
# 下载代码到本地
git clone https://github.com/uabjs/uab-weapp.git
cd uab-weapp

# 安装依赖  (node版本：v16.20.xx，  yarn版本：1.22.19)
yarn
```
## 项目文档
[https://uabjs.github.io/uab-weapp](https://uabjs.github.io/uab-weapp)
```sh
# 启动文档
yarn run dev

# 打包文档：（输出目录：site-dist）
yarn run release:site
```

## 打包项目：
```sh
yarn run build:lib
```


## 微信开发者工具打开组件库案例
```sh
# 1. 启动文档
yarn run dev

# 2. 用微信开发者工具打开 example 文件夹
```
